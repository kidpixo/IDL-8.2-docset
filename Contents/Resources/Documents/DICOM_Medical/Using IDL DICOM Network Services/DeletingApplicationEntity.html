Deleting an Application Entity 

You can delete any custom Application Entity definitions that you have defined. You cannot delete the default Application Entities (those listed in Deleting an Application Entity  ). To delete an Application Entity, complete the following steps:

  1. Select the Application Entity to be deleted from the Existing Entities droplist.
  2. Click Delete .
  3. Confirm the deletion by clicking Yes .
  4. Click Save to record the changes or Cancel to undo the changes.
