I DLnetOGCWMS::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLnetOGCWMS::Cleanup method performs all cleanup on the object. All open network connections are closed when the IDLnetOGCWMS object is destroyed.

Syntax

Obj -> Cleanup

or

Obj ->[ IDLnetOGCWMS ::]Cleanup

or

OBJ_DESTROY, Obj

Arguments

None

Keywords

None

Version History

6.4
