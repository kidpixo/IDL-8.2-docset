IDLffXMLDOMNodeList

Superclasses | Creation | Properties | Methods | Version History

The IDLffXMLDOMNodeList class is a container for nodes (see IDLffXMLDOMNode and its subclasses).

Superclasses

None

Creation

See IDLffXMLDOMNodeList::Init .

Properties

Objects of this class have no properties of their own, but do have properties inherited from any superclasses.

Methods

This class has the following methods:

  • IDLffXMLDOMNodeList::Cleanup
  • IDLffXMLDOMNodeList::GetLength
  • IDLffXMLDOMNodeList::Init
  • IDLffXMLDOMNodeList::Item

In addition, this class inherits the methods of any superclasses.
