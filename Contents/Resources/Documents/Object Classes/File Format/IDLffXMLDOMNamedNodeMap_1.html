IDLffXMLDOMNamedNodeMap::GetLength

Syntax | Return Value | Arguments | Keywords | Version History

The IDLffXMLDOMNamedNodeMap:: GetLength function method returns the number of nodes in the named node map.

Syntax

Result = Obj ->[ IDLffXMLDOMNamedNodeMap ::]GetLength()

Return Value

Returns an integer containing the number of nodes in the named node map.

Arguments

None

Keywords

None
