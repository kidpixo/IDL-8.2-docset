IDLffXMLDOMComment::Init

The IDLffXMLDOMComment:: Init function method initializes the object, but this class should not be instantiated directly. Instead, methods such as IDLffXMLDOMNode::GetFirstChild are used to create objects of all classes except IDLffXMLDOMDocument .

This class should not be subclassed by a user class because the classes that create objects, such as IDLffXMLDOMDocument , will not be aware of any user-defined classes.
