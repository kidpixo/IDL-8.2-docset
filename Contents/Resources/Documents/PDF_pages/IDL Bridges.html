IDL Bridges

Bridge technology allows applications in different programming languages or environments share information: for example, between IDL and Java. With bridge technology, you can use an application that manipulates data in its native language by calling on objects and processes from IDL. See the Java IDL Bridge Routines .

You can find additional information in the IDL Connectivity Bridges guide, located in the help\pdf directory of your IDL installation. This guide includes topics such as:

  • IDL Import and Export Briges
  • Importing into IDL Using Java Objects
  • ActiveX and COM
  • Exporting from IDL using the Export Bridge Assistant
  • Using Exported Java or COM objects
  • Writing IDL Objects
  • Creating Custom Objects
  • IDL Java Object API
