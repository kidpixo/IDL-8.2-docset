H5T_GET_EBIAS

The H5T_GET_EBIAS function returns the exponent bias of a floating-point type.

Syntax

Result = H5T_GET_EBIAS( Datatype_id )

Return Value

Returns the exponent bias.

Arguments

Datatype_id

An integer representing the datatype’s identifier to be queried.

Keywords

None
