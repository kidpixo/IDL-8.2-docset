H5S_SELECT_NONE

The H5S_SELECT_NONE procedure resets the dataspace selection region to include no elements.

Syntax

H5S_SELECT_NONE, Dataspace_id

Arguments

Dataspace_id

An integer representing the dataspace’s identifier to be reset.

Keywords

None

Version History

5.6
