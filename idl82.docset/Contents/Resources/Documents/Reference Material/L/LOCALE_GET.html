LOCALE_GET

Syntax | Return Value | Arguments | Keywords | Version History

The LOCALE_GET function returns the current locale (string) of the operating platform.

Syntax

Result = LOCALE_GET( )

Return Value

Returns a string containing the current operating platform locale.

Arguments

None

Keywords

None

Version History

5.2.1
