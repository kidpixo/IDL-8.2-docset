IDLgrPolygon::GetProperty

Syntax | Arguments | Keywords | Version History

The IDLgrPolygon:: GetProperty procedure method retrieves the value of the property or group of properties for the polygons.

Syntax

Obj ->[ IDLgrPolygon ::]GetProperty[, PROPERTY=variable ]

Arguments

There are no arguments for this methods.

Keywords

Any property listed under IDLgrPolygon Properties that contains the word “Yes” in the “Get” column of the properties table can be retrieved using this method. To retrieve the value of a property, specify the property name as a keyword set equal to a named variable that will contain the value of the property.

Any keyword not recognized is passed to this object’s superclass.

Version History
