IDLffMrSID

Superclasses | Creation | Properties | Methods | Version History

An IDLffMrSID object class is used to query information about and load image data from a MrSID ( . sid ) image file.

Note: See IDL Feature Support for a list of supported platforms.

None

  • IDLffMrSID::GetProperty
