IDLffXMLDOMNode::GetNodeName

Syntax | Return Value | Arguments | Keywords | Version History

The IDLffXMLDOMNode:: GetNodeName function method returns the name of the node, depending on the calling node’s subclass type.

Syntax

Result = Obj ->[ IDLffXMLDOMNode ::]GetNodeName()

Return Value

Returns a scalar string whose value depends on the subclass type:
