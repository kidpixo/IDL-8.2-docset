IDLffXMLDOMDocument::GetProperty

The IDLffXMLDOMDocument:: GetProperty procedure method is used to get the values of various properties of the document object.

Syntax

Obj- >[ IDLffXMLDOMDocument ::]GetProperty [, PROPERTY=variable ]

Arguments

None

Keywords

Any property listed under IDLffXMLDOMDocument Properties that contains the word “Yes” in the “Get” column of the property table can be retrieved using this method. To retrieve the value of a property, specify the property name as a keyword set equal to a named variable that will contain the value of the property.
